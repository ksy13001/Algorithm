## 1. 템플릿 디렉토리 생성
```
  #mysite/config/setting.py
  
  TEMPLATES = [
    { 
      #...
      
      'DIRS' : [BASE_DIR / 'templates'], #BASE_DIR = c:/project/mysite
      
      #...
    }
  ]
  ```
 ```
  (mysite) c:/project/mysite> mkdir templates
 ```
 
 -> 앞으로 앱들의 템플릿은 c:/project/mysite/templates 에서 관리  
 -> pybo 앱 템플릿 -> c:/project/mysite/templates/pybo
